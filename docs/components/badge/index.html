Badge

A compact pill for statuses, counts, and labels.

Import

import { Badge } from "@farmui/core";

Usage

Every example has a CSStab — that’s the real, complete stylesheet for the component. It’s plain, static CSS: nothing runs in the browser.

Variants

Four visual styles for different emphasis levels.

0 kb runtime
FilledLightOutlineDot

Colors

Semantic colors map to design tokens.

0 kb runtime
PrimaryGrayDangerWarningInfo

Sizes

0 kb runtime
SmallMediumLarge

The dot variant shows a colored status dot before the label.

PropTypeDefaultDescription

leftSectionReactNode—Content rendered before the label.
rightSectionReactNode—Content rendered after the label.
