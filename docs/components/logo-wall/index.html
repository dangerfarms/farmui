Blocks

LogoWall

A muted, wrapping row of customer logos that lift out of grayscale on hover.

Import

import { LogoWall } from "@farmui/blocks";

Usage

Every example has a CSStab — that’s the real, complete stylesheet for the component. It’s plain, static CSS: nothing runs in the browser.

With caption

Pass inline SVG via each logo's `node` prop for crisp, self-contained marks. Every logo still needs an `alt` label.

0 kb runtime

Trusted by leading teams

From image URLs

When you have hosted image files, pass `src` instead of `node`. `alt` is required either way.

0 kb runtime

Props

PropTypeDefaultDescription
logosLogo[]—The logos to render, where Logo = { alt: string; src?: string; node?: ReactNode }. Provide `src` for an <img>, or `node` for a custom mark (e.g. inline SVG); `alt` is always required. Required.
titleReactNode—Small centered caption above the logos, e.g. "Trusted by leading teams".
...othersHTMLAttributes<HTMLElement>—All native <section> props are forwarded.
