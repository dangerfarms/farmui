Space

An empty box used to add whitespace between elements.

Import

import { Space } from "@farmui/core";

Usage

Every example has a CSStab — that’s the real, complete stylesheet for the component. It’s plain, static CSS: nothing runs in the browser.

Set h to add vertical space between two blocks.
