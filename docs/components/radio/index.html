Props

PropTypeDefaultDescription
labelReactNode—Radio: label rendered next to the control.
descriptionReactNode—Radio: helper text rendered under the label.
size"sm" | "md" | "lg""md"Radio / RadioGroup: control size.
RadioGroup.labelReactNode—Group label (wired via aria-labelledby).
RadioGroup.descriptionReactNode—Helper text rendered under the group label.
RadioGroup.errorReactNode—Error message; marks the group invalid.
RadioGroup.namestring—Shared name for all radios (auto-generated if omitted).
RadioGroup.valuestring—Controlled selected value (pair with onChange).
RadioGroup.defaultValuestring—Initial selected value for uncontrolled usage.
RadioGroup.onChange(value: string) => void—Fires with the newly selected value.
RadioGroup.orientation"vertical" | "horizontal""vertical"Layout direction of the options.
RadioGroup.dataArray<string | { value; label? }>—Build options from an array instead of children.
...othersInputHTMLAttributes—All native <input type="radio"> props are forwarded.
