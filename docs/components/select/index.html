Select

A styled wrapper around a native select — accessible and zero-JS.

Import

import { Select } from "@farmui/core";

Usage

Every example has a CSStab — that’s the real, complete stylesheet for the component. It’s plain, static CSS: nothing runs in the browser.

Props

PropTypeDefaultDescription
labelReactNode—Field label rendered above the select.
descriptionReactNode—Helper text rendered below the label.
errorReactNode—Error message; puts the field in an invalid state.
size"sm" | "md" | "lg""md"Control size (height, padding, font size).
radius"sm" | "md" | "lg" | "xl" | "full""md"Border radius token.
withAsteriskboolean—Show a required asterisk next to the label.
placeholderstring—Non-selectable prompt shown as the first, empty option.
dataArray<string | { value: string; label: string }>—The options to render.
...othersSelectHTMLAttributes—All native <select> props are forwarded.
