Layout

Grid

A responsive 12-column layout whose spans adapt to the Grid's own width.

Import

import { Grid, GridCol } from "@farmui/core";

Usage

Every example has a CSStab — that’s the real, complete stylesheet for the component. It’s plain, static CSS: nothing runs in the browser.

Basic columns

Each GridCol spans a number of the 12 columns.

0 kb runtime
span 4
span 4
span 4

Responsive spans

Columns adapt to the Grid's width via container queries — full width by default, half at md, a third at lg.

0 kb runtime
span 12 · md 6 · lg 4
span 12 · md 6 · lg 4
span 12 · md 6 · lg 4

Auto columns

Use span="auto" to size a column to its content while siblings fill the rest.

0 kb runtime
auto
span 6
auto

Gutter

Control the gap between columns and rows.

0 kb runtime
span 6
span 6

Props

PropTypeDefaultDescription
columnsnumber12Total number of columns.
gutter"xs" | "sm" | "md" | "lg" | "xl" | number | string"md"Gap between columns and rows.
align"start" | "center" | "end" | "stretch" | "baseline"—align-items for the grid rows.
justify"start" | "center" | "end" | "between" | "around" | "evenly"—justify-content for the grid.
...othersHTMLAttributes<HTMLDivElement>—All native <div> props are forwarded.
GridCol spannumber | "auto"full widthColumns to span, or "auto" to size to content.
GridCol smnumber—Span once the Grid is ≥ 36rem wide (container query).
GridCol mdnumber—Span once the Grid is ≥ 48rem wide.
GridCol lgnumber—Span once the Grid is ≥ 62rem wide.
GridCol xlnumber—Span once the Grid is ≥ 75rem wide.
