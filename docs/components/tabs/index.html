Navigation

Tabs

Switch between related panels of content in the same view.

Import

import { Tabs } from "@farmui/core";

Usage

Every example has a CSStab — that’s the real, complete stylesheet for the component. It’s plain, static CSS: nothing runs in the browser.

Basic

Uncontrolled via defaultValue. Arrow keys move between tabs.

0 kb runtime
Update your name and email address.
Change your password and enable 2FA.
Choose how you want to be notified.

With icons

Use leftSection to place an icon or emoji before the label.

Props

PropTypeDefaultDescription
defaultValuestring—Value of the tab active by default (uncontrolled).
valuestring—Controlled active tab value.
onChange(value: string) => void—Called with the new value when the active tab changes.
TabsTab valuestring—Unique value linking a tab to its panel (required).
TabsTab leftSectionReactNode—Content (icon/emoji) rendered before the tab label.
TabsTab disabledboolean—Disable the tab and skip it in keyboard navigation.
TabsPanel valuestring—Value of the tab this panel belongs to (required).
...othersHTMLAttributes—All native <div>/<button> props are forwarded.
