Data display

Avatar

An image, initials, or fallback glyph representing a user.

Import

import { Avatar, AvatarGroup } from "@farmui/core";

Usage

Every example has a CSStab — that’s the real, complete stylesheet for the component. It’s plain, static CSS: nothing runs in the browser.

Image

Pass a src to render a cover-fit image.

0 kb runtime

Initials

With no image, initials are derived from name. Colors map to tokens.

0 kb runtime
JDSRAO

Sizes

Use a token or an explicit pixel number.

0 kb runtime
JDJDJDJD

Group

AvatarGroup overlaps children with a surface-colored ring.

0 kb runtime
JDSRAO+5

Props

PropTypeDefaultDescription
srcstring—Image source. When set, renders an <img>.
altstring—Alt text for the image (falls back to name).
namestring—Person's name; used for initials and as image alt.
size"sm" | "md" | "lg" | number"md"Size token or explicit pixel size.
radius"sm" | "md" | "lg" | "xl" | "full""full"Border radius token.
color"primary" | "gray" | "danger" | "warning" | "info""primary"Background color for the initials/fallback state.
...othersSpanHTMLAttributes—All native <span> props are forwarded.
