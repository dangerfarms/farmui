Blocks

BentoGrid

An asymmetric grid of feature cards where span-2 items stretch across two columns.

Import

import { BentoGrid } from "@farmui/blocks";

Usage

Every example has a CSStab — that’s the real, complete stylesheet for the component. It’s plain, static CSS: nothing runs in the browser.

Mixed spans

Give a standout card `span: 2` to make it twice as wide as the rest.

0 kb runtime

Props

PropTypeDefaultDescription
itemsBentoItem[]—The cards to render, where BentoItem = { title: ReactNode; description?: ReactNode; icon?: ReactNode; span?: 1 | 2 }. Required.
eyebrowReactNode—Small overline above the section title.
titleReactNode—Section heading.
descriptionReactNode—Supporting paragraph under the section title.
...othersHTMLAttributes<HTMLElement>—All native <section> props are forwarded.
