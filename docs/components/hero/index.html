Props

PropTypeDefaultDescription
titleReactNode—Main headline, rendered as an <h1>. Required.
eyebrowReactNode—Small overline shown above the title.
descriptionReactNode—Supporting paragraph under the title.
actionsBlockAction[]—Call-to-action buttons, where BlockAction = { label: string; href?: string; variant?: "filled" | "light" | "outline" | "subtle" }.
mediaReactNode—Optional media beside the copy (image, illustration, a Card…).
align"start" | "center""center" (or "start" when media is present)Layout alignment of the content.
...othersHTMLAttributes<HTMLElement>—All native <section> props are forwarded.
