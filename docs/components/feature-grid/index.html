Blocks

FeatureGrid

A responsive grid of icon / title / description features under an optional section header.

Import

import { FeatureGrid } from "@farmui/blocks";

Usage

Every example has a CSStab — that’s the real, complete stylesheet for the component. It’s plain, static CSS: nothing runs in the browser.

The classic three-up feature section with a header.

Props

PropTypeDefaultDescription
featuresFeature[]—The features to render, where Feature = { icon?: ReactNode; title: ReactNode; description?: ReactNode }. Required.
eyebrowReactNode—Small overline above the section title.
titleReactNode—Section heading.
descriptionReactNode—Supporting paragraph under the section title.
minChildWidthstring"16rem"Minimum card width; the grid wraps responsively.
...othersHTMLAttributes<HTMLElement>—All native <section> props are forwarded.
